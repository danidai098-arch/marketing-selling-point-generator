# Marketing Selling Point Generator

> 🎯 **让产品价值被看见，让用户心动并行动**

---

## 📖 项目简介

在信息爆炸的时代，好产品很多，但能被记住的产品很少。

太多优秀的产品，因为没能找到那个"让用户心动的理由"，最终淹没在市场的喧嚣中。
**Marketing Selling Point Generator** 就是为解决这个问题而生的——它不是简单的文案生成工具，而是你身边的营销策略顾问。

它懂得如何从纷繁复杂的产品信息中，提炼出最打动人心的价值点；它知道如何用情绪钩子抓住用户的注意力；它更明白，营销的本质不是"说清楚"，而是"被记住"。

**找卖点、定卖点、写卖点**——三个环节，一套工具，让每一个产品都能讲好自己的故事。

---

## ✨ 核心功能

### 🔍 找卖点 — 多维度扫描，发现隐藏价值

产品卖点往往藏在细节里。通过多种分析工具，帮你从不同角度扫描产品特征：

- **属性解析工具**：结构化提取产品功能、性能、规格等硬性指标
- **竞品对比工具**：横向分析市场同类产品，识别差异化优势
- **用户评价分析**：从真实反馈中挖掘用户最在意的价值点
- **痛点映射工具**：关联产品特性与用户痛点，找到"解决什么问题"的答案

### 🎯 定卖点 — 策略排序，锁定核心价值

找到很多卖点，但不知道哪个最重要？通过智能评估工具，帮你做出选择：

- **差异化评分引擎**：量化每个卖点的竞争优势强度
- **受众匹配分析**：根据目标人群画像，筛选最契合的卖点
- **场景适配评估**：判断卖点在不同使用场景下的说服力
- **传播潜力预测**：评估卖点的记忆点和传播性

### ✍️ 写卖点 — 情绪驱动，产出打动人心的文案

好的卖点需要好的表达。通过文案生成工具，让价值被看见：

- **FAB 转化引擎**：将产品属性转化为用户利益点
- **情绪钩子设计**：设计能引发共鸣或好奇的表达方式
- **多平台适配器**：一键生成适配电商、社交、广告等不同平台的文案风格
- **A/B 版本生成**：输出多个文案变体，支持效果测试

## 🎛️ 配置选项

| 参数 | 说明 |
|------|------|
| `platform` | 目标平台：电商详情页 / 社交媒体 / 广告投放 |
| `style` | 文案风格：专业严谨 / 活泼有趣 / 高端简约 |
| `max_points` | 最大卖点数量（建议 3-5 个） |
| `include_emotion_hook` | 是否生成情绪钩子文案 |
| `include_cta` | 是否包含转化引导（CTA） |
